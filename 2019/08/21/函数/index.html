函数

发表于 2019-08-21

基本使用

函数是一个符号，它代表了一段代码片段
定义：
function 函数名(参数1，参数2){
函数体(代码片段)
}

调用函数

函数名(参数1，参数2)

注意：

1、函数名只能包含数字、字母、下划线和$，数字不能开头
2、函数定义时，代码并没有执行，在函数调用时，代码才执行
3、函数定义时的参数叫做形参，函数调用时的参数叫做实参，在函数调用时，会将实参赋值给对应位置的形参。

ES5中，变量是又作用域的：局部作用域，全局作用域
局部作用域：函数内的范围
全局作用域：整个页面都是访问的范围

局部变量：在函数内定义的变量，就是局部变量，该变量只能在顶i有它的函数中使用
全局变量：在函数外定义的变量，就是全局变量，在定义完该变量后，在任何位置都能访问

r是一个局部变量，所以在函数外无法访问r
为了能将函数内的执行结果传递给函数外，我们需要使用return 语句，将某个变量/表达式返回
r是该函数的返回值，也就是函数调用时其表达式的结果

在实参个数 > 形参个数的情况下，如何得到所有的实参？
在函数调用时，浏览器会将所有的实参放置在一个对象中
放置的顺序，就是实参传递的顺序
我们在函数内，可以通过 arguements 得到这个对象
这个对象有一个属性 arguements.length 代表实参的个数
访问每一个实参，可以通过序号访问，序号从0开始

函数声明提升：浏览器在访问JS代码时，会将函数的定义提升到标签的最上边

函数表达式：将匿名函数赋值给一个变量

递归

用递归解决问题：

  • 1、将解决的问题，要能够分解与当前问题类似的
  • 2、分解之后的子问题，要比当前问题更简单
  • 3、问题要有一个终止条件

数学函数

Math.round()、Math.cell()、Math.floor()
Math.floor() : 向下取整
Math.cell() : 向上取整
Math.round() : 四舍五入

Math.max()、Math.min()
Math.min() : 最小值
Math.max() : 最大值
Math.pow()、Math.sqrt()
Math.pow() : 平方
Math.sqrt() : 开方
Math.abs 取绝对值
Math.randonm() 返回一个[0,1]之间的小数

箭头函数

作用：定义匿名函数

箭头函数的写法

1、 规则一:箭头函数只能用赋值式写法，不能用声明式写法。

2、 规则二:如果参数只有一个，可以不加括号，如果没有参数或者参数多于一就需要加括号

3、 规则三:如果函数体只有一句话，可以不加花括号

4、 规则四:如果函数体没有括号，可以不写return，箭头函数会帮你return

箭头函数this中的this

1、 箭头函数的特性一 ：默认绑定外层this

在使用箭头函数时，箭头函数默认不会指向使用自己的this，而是会和外层的this保持一直，最外层的this就是window
2、 箭头函数的特性二:不能用call修改里面的this

window.setTimeout()中函数里的this默认是window，我们也可以通过箭头函数使它的this和外层的this保持一致.

函数obj.a没有使用箭头函数，因此它的this还是obj，而setTimeout里的函数使用了箭头函数，所以它会和外层的this保持一致，也是obj；如果setTimeout里的函数没有使用箭头函数，那么它打出来的应该是window对象。
